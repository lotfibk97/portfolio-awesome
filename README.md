# Personal Portfolio

A portfolio built in React and NextJS. Simple, clean and fast.

  
## Demo

[Live Preview](https://www.vipuljha.com)

## Run Project 

Run this project with Yarn

```bash 
  yarn && yarn run dev
```
    
## Deployment

To deploy this project run

```bash
  yarn build
```
